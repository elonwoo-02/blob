# Dock 栏设计规范

## 🎨 设计理念

**新拟物主义 + 微妙动效**

融合现代极简与拟物化的精妙平衡，打造一个既保留 macOS Dock 经典气质，又具有当代审美的导航体验。

### 核心原则

1. **微妙的深度感**：通过分层阴影而非粗糙的 border，创造精致的浮起效果
2. **玻璃态美学**：采用半透明背景 + 模糊效果，营造磨砂玻璃的高级感
3. **极简的几何**：圆角、竖线分隔、均匀间距，遵循 macOS 的设计语言
4. **动作即反馈**：每一个交互都伴随流畅的过渡动画，让用户感知到系统的响应

## 🎨 色彩系统

### 主色调
- **深灰色**：`#1a1a1a` - `#2d2d2d`
- **原因**：与 macOS 深色模式一致，显得高级且专业

### 背景
- **半透明黑色**：`rgba(0, 0, 0, 0.6)`
- **模糊效果**：`backdrop-blur-xl`
- **原因**：玻璃态效果，既能融入任何背景，又能突出内容

### 强调色
- **蓝色系**：`#0084ff` - `#0066cc`
- **原因**：Apple 官方配色，用于活跃状态和悬停效果

### 文本
- **白色**：`#ffffff`
- **浅灰色**：`#e5e5e5`
- **原因**：与深色背景形成高对比度，确保可读性

### 分隔线
- **默认**：`rgba(255, 255, 255, 0.2)`
- **悬停**：`rgba(0, 132, 255, 0.6)` 渐变

## 📐 尺寸规范

### Dock 栏整体
| 属性 | 值 | 说明 |
|------|-----|------|
| 高度 | 64px | 包括底部间距 |
| 宽度 | 自适应 | 根据内容数量 |
| 圆角 | 24px | `rounded-3xl` |
| 底部间距 | 24px | `mb-6` |
| 内边距 | 12px 16px | `px-4 py-3` |

### 图标
| 属性 | 值 | 说明 |
|------|-----|------|
| 尺寸 | 48px × 48px | 标准大小 |
| 圆角 | 12px | `rounded-xl` |
| 间距 | 8px | 图标之间 |
| 悬停放大 | 1.3x | `scale-125` |

### 竖线分隔
| 属性 | 值 | 说明 |
|------|-----|------|
| 宽度 | 1px | `w-px` |
| 高度 | 32px | `h-8` |
| 左右间距 | 8px | `mx-2` |
| 颜色 | `rgba(255, 255, 255, 0.2)` | 默认 |

## 🎬 动画规范

### 隐藏/显示动画
- **触发**：3 秒无操作 / 鼠标移至底部
- **方向**：向下滑出 / 向上滑入
- **时长**：300ms
- **缓动**：`cubic-bezier(0.4, 0, 0.2, 1)` (Material Design 标准)
- **距离**：`translateY(120%)`

### 图标悬停动画
- **放大**：`scale(1.3)`
- **阴影**：从 `shadow-lg` 到 `shadow-2xl`
- **时长**：150ms
- **缓动**：`ease-out`

### 分隔线光晕
- **从**：`rgba(255, 255, 255, 0.2)`
- **到**：`rgba(0, 132, 255, 0.6)` 渐变
- **时长**：200ms
- **缓动**：`ease-out`
- **光晕**：`0 0 8px rgba(0, 132, 255, 0.4)`

## 🔲 阴影系统

### 默认阴影
```css
box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.3),
            0 10px 10px -5px rgba(0, 0, 0, 0.04);
```

### 悬停阴影
```css
box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.5);
```

### 图标阴影
```css
filter: drop-shadow(0 4px 6px rgba(0, 0, 0, 0.1));
```

## 🔤 排版系统

### 工具提示
- **字体**：系统字体栈
- **大小**：12px (`text-xs`)
- **粗细**：500 (`font-medium`)
- **颜色**：白色
- **背景**：深灰色 (`#111827`)

### 标签
- **字体**：系统字体栈
- **大小**：14px (`text-sm`)
- **粗细**：400 (normal)
- **颜色**：白色

## 📱 响应式设计

### 桌面 (≥ 1024px)
- 完整显示，无调整
- 所有功能正常

### 平板 (768px - 1023px)
- Dock 栏宽度减小至最大 500px
- 图标尺寸保持不变

### 手机 (< 768px)
- 建议隐藏 Dock 栏
- 改用传统底部导航栏

## ♿ 无障碍性

### 键盘导航
- Tab 键可以在项目间导航
- Enter 键激活链接

### 屏幕阅读器
- 使用语义化 HTML (`<nav>`, `<a>`)
- 提供 `aria-label` 属性

### 颜色对比度
- 文字与背景对比度 ≥ 4.5:1
- 符合 WCAG AA 标准

## 🎯 交互细节

### 鼠标进入 Dock 项目
1. 图标放大至 1.3 倍
2. 阴影增强
3. 显示工具提示
4. 重置隐藏计时器

### 鼠标离开 Dock 项目
1. 图标缩小回原大小
2. 阴影恢复
3. 隐藏工具提示
4. 启动 3 秒隐藏计时器

### 鼠标移至屏幕底部
1. 检测鼠标在底部 100px 区域
2. 立即显示 Dock 栏
3. 清除隐藏计时器

### 点击 Dock 项目
1. 导航到指定链接
2. 外部链接在新标签页打开
3. 内部链接在当前标签页打开

## 🔧 实现建议

### CSS 类命名
- 使用 Tailwind CSS 工具类
- 避免自定义 CSS 类
- 保持代码简洁

### 性能优化
- 使用 `will-change: transform, opacity` 优化动画
- 避免频繁的 DOM 操作
- 使用事件委托处理鼠标事件

### 浏览器兼容性
- 使用 `-webkit-` 前缀处理 Safari
- 测试 Chrome, Firefox, Safari, Edge
- 不支持 IE 11

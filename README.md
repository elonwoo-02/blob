# Dimensional Debris

Astro-based personal site with a macOS-style Dock, blog, tags, and markdown post layout.

## Features
- macOS-style floating Dock navigation with activity windows.
- Blog index with tag sidebar, heatmap view, and clickable cards.
- Nested tags with drawer-style navigation.
- Markdown post layout with chapter navigation and typography styling.

## Project Structure
```
/
├─ public/
├─ src/
│  ├─ components/
│  ├─ layouts/
│  ├─ pages/
│  └─ blog/
└─ package.json
```

## Commands
All commands run from the repo root:
- `npm install` — install dependencies
- `npm run dev` — start dev server at `http://localhost:4321`
- `npm run build` — build to `./dist`
- `npm run preview` — preview the build
